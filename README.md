# MathLab + Yew

## MathLab

> A Powerful Math Library for Rust

## Install

Run the following Cargo command in your project directory:

```shell
cargo add mathlab
```

or

Add the following line to your `Cargo.toml` file with the specified version:

```toml
mathlab = "MAJOR.MINOR.PATCH"
```
